Featured project

HD DVD Store

A React storefront for HD DVDs. The application utilises TypeScript to ensure type safety and clean, deployable code. It also uses BootStrap to create a responsive UX as the user goes between the homepage, storefront, shopping cart and a page on the history of the short-lived disc format.

React Router is used to navigate between the different pages. It also uses the Context API, various hooks (including a custom one), JSON, and local storage to manage the state of the shopping cart.

Projects

The following are eight Frontend projects I have worked on. Three of them were tasked by Meta to receive their Frontend Developer certification, with the others completed to further skills in React, TypeScript, Bootstrap etc.

Hangman

Hangman

The classic Hangman game created in React and TypeScript via Vite. It also uses CSS modules that applies responsive design.

The user has access to both keyboard and mouse inputs when guessing letters. The application itself uses various hooks like useCallback, useEffect and useState for its logic.

GitHub Website

PlayStation Productions

PlayStation Productions

A React homepage for PlayStation's TV and Movie division, which imports Bootstrap for the CSS. This project was all about seeing how the two technologies work together, as React Bootstrap differs from the regular library.

The homepage also uses higher-order functions like map to have the code adhere to DRY.

GitHub Website

Calculator App

Calculator App

A working React calculator. Tasked by Meta to test my knowledge of that chapter's React concepts.

The useState and useRef hooks, as well as a controlled form, are used for functionality. I also coded custom CSS to create a realistic looking calculator for a more fun user experience.

GitHub Website

Portfolio Page for Pete

Portfolio Page for Pete

A portfolio page for Meta's fictional client. Created in React with Create React App, the page also incorporates Chakra UI, Formik and Yup.

A particularly rewarding project to complete. It drilled in concepts like how to structure a CRA project, how to use 3rd-party libraries in React, breaking the page down into components and prop handling.

GitHub Website

Movie Recommender

Movie Recommender

This project calls The Movie Database API to fix choice paralysis. The user picks a genre and a movie fitting that selection will have its information displayed. They then interact with the page to like or dislike the API's recommendation.

This project was created with the sole goal of getting hands-on experience with Asynchronous JavaScript. Particularly with ES8 Async/Await.

GitHub Website

XGS Publishing

XGS Publishing

A Bootstrap homepage for Xbox's publishing division. The project includes numerous Bootstrap components like modals, forms and cards. Plus some vanilla JavaScript and the Mapbox API.

As my first Bootstrap project, this one was particularly rewarding because it helped me achieve my goal of learning one of the popular CSS libraries.

GitHub Website

Live Music Locator

Live Music Locator

This project uses vanilla HTML and CSS to create a non-scrolling homepage with a video background. Live Music Locator also features the simplistic UX these modern layouts are known for.

The video was professionally edited in Premiere Pro, and set up so that user accessing the CSS can add their own color overlay to suit their tastes.
